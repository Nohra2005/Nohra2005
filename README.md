### 🔍 Locus — Retail Visual Search Platform
**AUB · AI in Industry** &nbsp;|&nbsp; *In Progress*

A computer vision-powered **Progressive Web App (PWA)** that lets shoppers upload any photo and instantly find matching products across nearby stores. Built as a full microservices system — from image ingestion to semantic vector search — deployed on the cloud.

**How it works:** A user uploads a photo → YOLOv8 detects and crops the item → CLIP encodes it into a semantic embedding → Qdrant finds the closest matches in the product vector database → results are returned in real time.
